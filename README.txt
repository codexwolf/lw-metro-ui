LW Metro UI is a WordPress theme inspired by the Metro design of Windows Phone 7 and Windows 8.

LW Metro UI is a child theme for the Thematic Theme Framework.
Therefore, you must have Thematic installed first before activating this theme.
http://themeshaper.com/thematic/


INSTALL:
Simply copy the "lw-metro-ui" folder to your /wp-content/themes folder.
Then activate the theme through your WordPress admin.
